# Ulfron tree

Ulfron tree is a project to create a family tree of the Ulfron Family. A family from a RP i have with a friend. 

To see it you just have to download the last .zip in the release, unzip it and launch the .exe folder.

# WARNING

This is a early access of the Ulfron Tree, you have just access to the beta. The Gold version is not ready yet. This project isn't complete yet.
